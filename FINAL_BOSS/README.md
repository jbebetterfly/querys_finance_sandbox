# Final Boss

Query consolidada de P&L y métricas operativas para Betterfly. Integra ingresos, costos y KPIs de todas las líneas de negocio en una sola vista.

## Cobertura

### Ingresos
- **Employee Benefit** — Logos y Members incluidos
- **Affinity (Subscription)** — Logos y Members incluidos
- **Rewards**
- **Otros negocios** — ingresos no recurrentes

### Costos
- **Subscription**
- **Rewards**

### Gastos
- **SG&A** — excluye Severance, Uncollectible Accounts y Non-Operational

### Métricas operativas
- **Booked MRR** — Chile, México, España
- **Onboarding**
- **MAU**
